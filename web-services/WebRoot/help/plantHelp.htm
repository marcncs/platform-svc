第一章 标签打印

1.1数据上传

1.1.1 杭州工厂SAP数据上传

       1.通过SAP系统创建订单。在订单完成后SAP系统会把SAP订单传输到RTCI系统中 点击进入标签打印→任务日志 进入“任务日志”查看页面 查看任务处理状态

 

2.任务处理状态为【正确处理】。点击进入标签打印→打印任务，进入“打印任务”页面。查看打印任务列表，打印任务已生成，且小包装标签状态为【未生成】。

 

1.1.2杭州工厂手工数据上传

       1.点击进入标签打印→数据导入 进入“手工上传数据页面。

2.选择的工厂类型为“杭州工厂”。

3.点击【浏览】 选择文件（XML文件格式内容正确）。

4.点击【上传】 弹出提示信息：“文件上传成功， 请通过上传日志记录查看处理结果!”。文件上传成功。

备注：手工上传文件日期格式（yyyyMMdd,dd-MM-yyyy,yyyy-MM-dd, MM/dd/yyyy或者yyyy/MM/dd）

 

1.1.3分装厂手工数据上传

和杭州工厂手工上传文件操作步骤一致。

1.2.箱码标签打印

1.通过客户端登录，进入打印界面。

 

 

打印界面

2.进入“拜耳打印系统→标签打印列表” 点击界面中【显示】显示未打印任务列表。

选择需打印任务 点击【打印】进入“标签打印详情”界面

3.设置商品【分装日期】、【线号】、【打印份数】、【语言】。 点击【确认打印信息】 打印准备就绪。

 

4.点击【打印】 弹出提示信息：“共XX条数据 是否需要打印” 点击【确定】 开始打印。

5.打印成功 状态日志中显示打印信息， 打印顺序按托盘和条码的大小进行升序排列。

1.3.箱码标签重新打印

1.进入“拜耳打印系统→标签打印列表”。点击界面中【重新打印】进入“标签重新打印界面”。点击界面中【显示】列表中显示已打印标签任务

2. 在列表中选择标签打印任务 点击【打印】，进入“重打标签详情”页面。

3.在“重打标签详情”页面中，可自定义打印范围 如【序列号】、【SSCC】等。序列号既可以连续也可特别制定。打印范围确认后，点击【预览】，即可查看需打印内容。

4. 在“重打标签详情”页面中，确认打印范围后 选择【重打原因】、【标签份数】和【标签语言】，点击【重新打印】。弹出提示信息：“共XX条数据 是否需要打印”。 点击【确定】 标签开始重新打印。

1.4.打印历史记录查看

1.进入“拜耳打印系统→标签打印列表”，点击界面中【打印历史记录】进入“标签打印记录列表”。点击界面中【显示】，列表中显示已打印标签任务。

2.进入“标签打印记录列表”，在列表中选择已打印任务点击【查看】，进入“打印任务详情”界面，显示以打印任务详情及箱码信息。

3. 在“打印任务详情”界面中，点击【打印记录】，可查看该任务打印详细记录。

1.5.商品小包装码生成及下载

1.登录BCS系统。点击进入“标签打印→打印任务“进入打印任务页面，查看已打印任务，小包装码已生成。

2. 选择打印任务 点击该任务【小包装标签】栏中【下载】 下载小包装标签 小包装码下载成功。

小包装码生成文件

1.6.商品通用码生成

1. 点击进入标签打印→打印任务 进入“打印任务”页面，选择已打印任务 点击【生成通用码】 进入“生成通用码”页面。

 

2. 在“生成通用码”页面中，填写【生成数量】 点击【确定】，弹出提示信息：“通用码打印任务添加成功，请通过日志查看处理结果”。

3. 进入“打印任务”页面，选择已打印任务，点击【通用码】查看通用码生成记录和详情

4.点击【下载】，查看生成通用码详细内容

通用码生成文件

1.7.暗码上传

1.点击进入“生产数据→暗码上传 进入“暗码上传”页面。

2. 点击页面中【暗码上传】进入“上传暗码”页面

3. 点击【浏览】选择上传文件（文件格式为TXT格式 文件内容格式正确）。点击【确定】，文件上传成功。

4. 点击进入“生产数据→暗码上传日志，查看已上传文件解析结果。

5. 点击进入“生产数据→暗码上传 进入“暗码上传”页面，在【小包装码】输入框中输入想查询的商品小包装码，点击【搜索】，可查看到已上传的小包装码对应的暗码信息。
